# Aurora Flow

Two million GPU particles riding a real-time fluid simulation, in a single dependency-free HTML file.

**[Live demo](https://lemonbanan4.github.io/aurora-flow/)** — move your cursor to stir the fluid, click for a burst of color.

- Sound-reactive mode: the mic drives frequency-band jets and beat-detection bursts
- Navier-Stokes solver (advection, pressure projection, vorticity confinement) in WebGL2 fragment shaders
- 2,097,152 particles advected by the velocity field, updated and rendered entirely on the GPU
- No libraries, no build step — one `index.html`
